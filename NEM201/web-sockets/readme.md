## Uses of web-sockets
1. chatting
2. payments
3. gaming (multiplayer)
4. live streaming (not video) for video -> WebRTC
5. comments
6. financial ticker
7. match scores
8. weather
9. social media feeds
10. order tracking

## Advantages
1. real time communication
2. don't need to refresh page
3. faster, cheaper

## Disadvantages
1. text based
